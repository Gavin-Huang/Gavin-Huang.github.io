Ubuntu下showsocks chrome 配置说明

本教程示例在ubuntu的环境下安装shadowsocks并且配置chrome

在ubuntu下安装shadowsocks并且配置chrome

  • 因为要安装ubuntu环境下的tensorflow,需要连接一些外网的软件包，所以需要搭建翻墙shadowsocks,我搭建成功了，中途遇到很多坑，写下这篇教程来记录一下配置。

    安装chrome

    使用ctrl+alt+t进入terminal
  • 将下载源加入到系统的源列表:
    • sudo wget http://www.linuxidc.com/files/repo/google-chrome.list -P /etc/apt/sources.list.d/
  • 导入谷歌软件的公钥:
    • wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
  • 用于对当前系统的可用更新列表进行更新:
    • sudo apt-get update
  • 谷歌 Chrome 浏览器（稳定版）的安装:
    • sudo apt-get install google-chrome-stable
  • 启动chrome:
    • /usr/bin/google-chrome-stable

      安装shadowsocks-qt5

  • 安装好chrome后，再安装shadowsocks-qt5
  • 依次使用以下命令行：
    • sudo add-apt-repository ppa:hzwhuang/ss-qt5
    • sudo apt-get update
    • sudo apt-get install shadowsocks-qt5

      配置shadowsocks

  • 安装完成后在软件中心搜索shadowsocks就可以看到相关图标，点击图标就可以进行配置
  • 新建服务器，填写server地址、密码等参数，（重要：将连接类型设置为sock5才可以进行连接，将debug打开可以看到更详细的错误）
  • 完成后点击connect,查看日志，无报错说明配置良好

    使用命令行启动chrome

    在terminal里使用google-chrome --proxy-server="sock5://127.0.0.1:1080"命令行打开chrome,才可以上外网，这一步很重要！
